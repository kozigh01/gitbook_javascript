### Notes

#### **Responsive Breakpoints, based on screen width:**

* \(xs\) — screen width &lt; 576px \(This is the “default” tier\)
* sm — screen width ≥ 576px
* md — screen width ≥ 768px
* lg— screen width ≥ 992px
* xl — screen width ≥ 1200px
